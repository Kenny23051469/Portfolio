Kenny's Travel List

About Kenny's Travel List

Kenny's Travel List is a simple travel list website which allows users to add items into a list, it features a way for users to mark an item as "Packed" which will then update the status bar at the bottom with the percentage packed.

Target Audience

Kenny's Travel List caters to users looking for a simple and easy to use Travel List website to manage their belongings before a trip.
