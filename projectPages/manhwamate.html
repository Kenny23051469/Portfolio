ManhwaMate

About ManhwaMate

ManhwaMate is a React Native mobile applications which allow users to select a predefined list of manhwas to add into a read list or add their own manhwas to the read list.

Target Audience

ManhwaMate's target audience is anyone who is interested in reading manhwas and who wants to keep track of their read list.

Technologies Used

React Native
JavaScript
Figma
